Experience

It was satisfying and difficult to learn HTML, CSS, and JavaScript. At first, using semantic HTML to structure my website seemed simple, but managing the layout and styling needed more knowledge. JavaScript was essential for interactive features like the rain motion, while CSS capabilities like Flexbox and Grid made responsive designs possible. A significant portion of the project involved debugging, particularly when dealing with responsive elements and animations. For resolving layout problems, tools like Chrome Developer Tools were quite helpful.

Design Choices

This website was designed with a cyberpunk-inspired aesthetic, combining a sleek, futuristic feel with a user-friendly experience. The black background provides a modern, high-tech look while making the neon accents stand out. White text ensures strong contrast and readability, while the sans-serif font keeps everything clean and contemporary. The layout is structured using CSS Grid, which allows for a flexible and responsive design that adapts to different screen sizes. Each section is neatly arranged, creating a balanced and visually appealing experience. Navigation is simple and intuitive, with evenly spaced links that include hover effects to provide instant feedback. To make the site feel more interactive, subtle animations were added. The fade-in effect on page load creates a smooth transition, while buttons and navigation links respond dynamically to user actions. The digital rain animation, inspired by cyberpunk visuals, adds movement and depth without overwhelming the content. The website is also optimized for cross-browser compatibility, ensuring it functions smoothly across different devices and screen sizes. Overall, this design balances aesthetics, usability, and performance, creating an immersive and engaging experience.
