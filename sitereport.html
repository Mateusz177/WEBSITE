Experience

It was satisfying and difficult to learn HTML, CSS, and JavaScript. At first, using semantic HTML to structure my website seemed simple, but managing the layout and styling needed more knowledge. JavaScript was essential for interactive features like the rain motion, while CSS capabilities like Flexbox and Grid made responsive designs possible. A significant portion of the project involved debugging, particularly when dealing with responsive elements and animations. For resolving layout problems, tools like Chrome Developer Tools were quite helpful.

Design Choices

I chose a black background with neon-style accents because my design approach was motivated by cyberpunk aesthetics. A sense of speed and excitement is added by the background rain effect. For readability, I went with a crisp, contemporary sans-serif typeface, and user engagement is improved by hover effects and animations. In order to portray important portions in an orderly and aesthetically pleasing way, the grid pattern was used.
